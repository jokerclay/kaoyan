## 常量与变量
### 常量
```C
#include <iostream>

int main() {
    int a = 3; // 把 3 这个值复制到 a 里，
               // 3 是从何而来的 ？3 是在代码段中, 执行到 int a = 3
               // 会把代码段中的 3 复制到 a 中

}
```
* 整型：1, 15, 56,  -15, 0
* 实型：0.23， 4.56， -5.36
* 字符型： 'a', 'c', 'e'
* 字符串型："abc", "clay", ""(空字符串)

### 变量
    代表内存中一个特定属性的存储单元,可以改变
    命名:字母，数字，下划线, 区分大小写
    **先定义，后赋值**
### 符号常量
    * 不可赋值
    * 关键字 `define`
